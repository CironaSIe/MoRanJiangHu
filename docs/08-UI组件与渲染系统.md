## 3. XSS 风险

### 3.1 dangerouslySetInnerHTML 使用

**唯一位置**: `components/features/Chat/MessageRenderers.tsx:1243`

```tsx
dangerouslySetInnerHTML={{ __html: htmlContent }}
```

**清洗流程**：
1. `hooks/useGame/sendWorkflow.ts:1807-1821` 在写入 `htmlContent` 前根据内容类型选择：
   - 含 `<script>` → `htmlRenderMode: 'sandbox'` → 沙箱 iframe
   - 纯 HTML 美化 → `清洗HTML美化产出()` → DOMPurify 清洗 → `htmlRenderMode: 'purify'`
2. `utils/tavernSandboxBridge.ts:50-70` DOMPurify 配置：
   - `FORBID_TAGS: ['script', 'iframe', 'object', 'embed', 'applet', 'form']`
   - `FORBID_ATTR: ['onclick', 'onerror', 'onload', 'onmouseover', ...]`
3. 沙箱 iframe 使用 `sandbox="allow-scripts"`（不含 `allow-same-origin`）

**评估**：✅ 链路安全，但 DOMPurify 调用不够显式。

## 4. 内存泄漏

### 4.1 useEffect 清理缺失统计

| 文件 | useEffect 数量 | 有清理 | 缺失率 |
|------|----------------|--------|--------|
| `App.tsx` | 31 | ~12 | ~61% |
| `ImageManagerModal.tsx` | 27 | 1 | **96%** 🔴 |
| `InputArea.tsx` | 7 | 2 | ~71% |
| `NovelDecompositionSettings.tsx` | 14+ | 未知 | 需审查 |
| `ChatList.tsx` | 4 | 3 | ✅ 良好 |
| `SocialModal.tsx` | 5 | 1 | ~80% |

### 4.2 具体泄漏风险

**`ImageManagerModal.tsx`（最高风险）**：
- 27 个 `useEffect` 仅 1 个有清理
- 多个 effect 订阅外部状态未在清理时取消订阅
- 多个 effect 依赖 `useCallback` 但未用 `useRef` 保持最新引用

**`InputArea.tsx`**：
- line 246、282 无清理（纯状态同步，无害）
- line 771、801 仅做状态判断（空 if 分支），属于无用 effect

**`App.tsx`**：
- line 2910 Capacitor 返回键监听使用 async + cancelled 标志，模式正确
- line 634 `startOnlinePresenceHeartbeat()` 清理依赖函数内部实现

### 4.3 定时器泄漏

- `TopBar.tsx:55` setTimeout 长按计时器 ✅ 有清理
- `InputArea.tsx:236` setInterval ✅ 有 clearInterval
- `SandboxedCard.tsx:187` setInterval(measureHeight, 2000) ✅ 有清理

## 8. 无障碍性 — ❌ 严重不足

### ARIA 标签 — 🟡 覆盖不完整
- 部分按钮有 `aria-label` ✅
- 大量交互按钮缺少 `aria-label`/`aria-expanded`/`aria-controls`
- 模态框缺少 `role="dialog"` 和 `aria-modal="true"`
- 表单元素缺少 `htmlFor`/`id` 关联

### 键盘导航 — 🟡 基础支持
- Enter 发送消息 ✅
- Escape 关闭图片查看器 ✅
- 大部分自定义按钮无法 Tab 聚焦 ❌
- 缺少 `focus-visible` 样式（533 处 `outline-none` 仅 28 处 `focus-visible`）

### 屏幕阅读器 — ❌ 几乎无支持
- 仅 10 处 `sr-only`（都在文件上传 input 上）
- 无 `aria-live` 区域
- 无跳过导航链接

### 动效无障碍 — ❌ 完全缺失
- 未检测 `prefers-reduced-motion`
- 1268 处动画/过渡类名无降级方案
